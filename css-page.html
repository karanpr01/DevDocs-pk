    • CSS Cheat Sheet for Quick Reference

    1. What is CSS?



    • CSS (Cascading Style Sheets) is used to style and layout web pages â€” like changing colors, fonts, spacing, etc.

    • Works alongside HTML to create visually attractive websites.

    2. How to Add CSS


    Method Syntax Use Case
    Inline CSS 
    
<h1 style="color:red;">Hello</h1>
    		Quick styling for single elements
    Internal CSS 
    
<style>h1 { color: red; }</style>
    		Styling one HTML file
    External CSS 
    
<link rel="stylesheet" href="style.css">
    		Best practice for larger websites

    3. CSS Syntax Basics

    
selector{
    property: value;
}

    Example:

    
p{
    color: blue;
    font-size: 16px;
}


    4. CSS Selectors

    Selector Example Description
    Element Selector p{} Targets <p> tags
    Class Selector .classname {} Targets elements with class="classname"
    ID Selector #idname{} Targets elements with id="idname"
    Universal selector *{} Targets all elements
    Grouping Selector h1,h2,p{} Style Multiple elements
    Descendant Selector div p{} Targets <p> under <div>


    5. CSS Properties


    Text & Font


    
color:red;
font-size:18px;
font-weight:bold;
font-family:'Arial', sans-serif;
text-align: center;
text-decoration: underline;
text-transform: uppercase;
letter-spacing: 2px;
line-height: 1.5;



    Background


    
background-color: lightblue;
background-image: url('image.jpg');
background-repeat: no-repeat;
background-size: cover;
background-position: center;



    Box Model


    
margin: 20px;
padding: 15px;
border: 2px solid black;
width: 300px;
height: 200px;



    Borders


    
border: 1px solid black;
border-radius: 8px;



    Display & Position


    
display: block;
display: flex;
position: relative;
position: absolute;
top: 20px;
left: 10px;



    Flexbox


    
display: flex;
justify-content: center;
align-items: center;
flex-direction: column;
gap: 10px;



    Grid


    
display: grid;
grid-template-columns: repeat(3, 1fr);
grid-gap: 10px;



    Others


    
opacity: 0.8;
visibility: hidden;
overflow: scroll;
box-shadow: 0px 4px 6px rgba(0,0,0,0.1);
transition: all 0.3s ease;



    6. Pseudo-classes


    
a:hover {
  color: green;
}

input:focus {
  border: 2px solid blue;
}

li:first-child {
  color: red;
}

li:last-child {
  color: green;
}



    7.Pseudo-elements


    
p::first-letter {
  font-size: 2em;
}

p::before {
  content: 'Note: ';
  color: red;
}



    8. Media Queries (Responsive Design)


    
@media (max-width: 768px) {
  body {
    background-color: lightgrey;
  }
}

    Use: Make your Website Mobile Friendly!




    9. Important CSS Shorthand


    
margin: 10px 20px 30px 40px; /* top right bottom left */
padding: 20px;
border: 1px solid red;
background: url('img.jpg') no-repeat center/cover;
font: italic small-caps bold 16px/2 Arial, sans-serif;


    10. Modern CSS Features


    Variables


    
:root {
  --main-color: #3498db;
}

h1 {
  color: var(--main-color);
}



    Clamp Function


    
font-size: clamp(1rem, 2.5vw, 2rem);



    Aspect Ratio


    
img {
  aspect-ratio: 16 / 9;
}



    11. Useful Tips


    • Always use external CSS for maintainability.

    • Use reset.css or normalize.css to fix browser inconsistencies.

    • Prefer Flexbox and Grid for layout building.

    • Always use responsive units like em, rem, %, vw, vh.


    12. Conclusion


    This cheat sheet gives a quick "grab-and-go" summary for anyone building modern, responsive, and stylish websites using CSS3. Whether you're working professionally or learning, keep it handy!
