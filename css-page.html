    1. What is CSS?



    • CSS (Cascading Style Sheets) is used to style and layout web pages â€” like changing colors, fonts, spacing, etc.

    • Works alongside HTML to create visually attractive websites.

    2. How to Add CSS


    Method Syntax Use Case
    Inline CSS 
    
<h1 style="color:red;">Hello</h1>
    		Quick styling for single elements
    Internal CSS 
    
<style>h1 { color: red; }</style>
    		Styling one HTML file
    External CSS 
    
<link rel="stylesheet" href="style.css">
    		Best practice for larger websites

    3. CSS Syntax Basics

    
selector{
    property: value;
}

    Example:

    
p{
    color: blue;
    font-size: 16px;
}


    4. CSS Selectors

    Selector Example Description
    Element Selector p{} Targets <p> tags
    Class Selector .classname {} Targets elements with class="classname"
    ID Selector #idname{} Targets elements with id="idname"
    Universal selector *{} Targets all elements
    Grouping Selector h1,h2,p{} Style Multiple elements
    Descendant Selector div p{} Targets <p> under <div>


    5. Input Types(HTML5 Forms)

    
<input type="email" placeholder="Enter your email">
<input type="date">
<input type="color">
<input type="range" min="0" max="100">
    • Collecting Specific type of user input.
    • Browser auto-validates fields like email.


    6. Semantic HTML

    Sematic HTML=Correctly describing your content

    Example:

    
<article>
<h2> Blog Post Title </h2>
<p>Post Content Here...</p>
</article>

    Why Importan?

    • Better SEO.
    • Imporved accessibility.
    • Clean Code.


    7. Meta Tags (SEO and Responsive Design)

    Sematic HTML=Correctly describing your content

    Example:

    
<meta name="description" content="Learn HTML quickly with this cheat sheet!">
<meta name="keywords" content="HTML, web development, cheat sheet">
<meta name="author" content="Your Name">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
    • Enhance SEO.
    • Make your Website Mobile Responsive.
