    • HTML cheat sheet

    A Quick Reference guide for HTML elements, Showcasing example usage and common use case.

    1. What is Html?

    HTML(Hyper Text Markup Language) is the standard language for creating Webpages. It defines the Structure of content, while css and javascript handle styling ans behavior.

    2. Basic HTML Structure

    
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

    3. Common HTML and their use cases

    Heading tags <h1> to <h6>

    
<h1>Main Title </h1>
<h2>Section Title </h2>
<h3>Subsection Title </h3>
    • Structuring title and subtitles.
    • SEO Tips: use only one <h1> per page.


    Paragraph Tag <p>

    
<p>This is Paragraph text </p>
    • Writing Regular text


    Anchor Tag <a>

    
<a href="https://example.com" target="_blank"> Visit example </p>
    • Creating Links
    • traget="_blank" opens links in new tab.


    Image Tag <img>

    
<img src="image.png" alt="Description of image width="300">
    • Adding images.
    • Always add alt for accessibility.


    List Tags <ol>, <ul>, <li->

    
<ul>
<li>First item</li>
<li>Secong item</li>
</ul>

<ol>
<li>First item</li>
<li>Secong item</li>
</ol>
    • <ul>=Unordered (bullets).
    • <ol>=Ordered (numbers).


    Table Tags <table>, <tr>, <th>, <td>

    
<table>
<tr>
<th>Name</th>
<th>Age</th>
</tr>
<tr>
<td>Jhon</td>
<td>30</td>
</table>
    • Dispalyin tabular data.


    Form Tags 

    
<form action="/submit" methond="POST">
<input type="text" name="username" placeholder="Enter Username">
<input type="password" name="password" placeholder="Enter password">
<button type="submit">Submit</button>
